Variables y Tipos de Operadores

Antes de continuar, es aconsejable usar una funcion de javascript para ver los resultados, con esto nos referimos a conlsole.log()

La función console.log() en JavaScript se utiliza para imprimir mensajes o valores en la consola del navegador o en la terminal (si estás ejecutando código en Node.js). Es una herramienta de depuración fundamental que permite a los desarrolladores mostrar información mientras están construyendo, probando y depurando sus aplicaciones.

Algunas de las principales utilidades de console.log() son:

  • Depuración: console.log() es una forma rápida y sencilla de verificar el valor de variables y comprobar si el flujo de tu código es el esperado. Puedes imprimir el contenido de variables, objetos, arreglos o cualquier otra información relevante para entender cómo se está ejecutando tu código.
  • Mensajes de estado: Puedes usar console.log() para mostrar mensajes de estado o progreso en tu aplicación. Por ejemplo, puedes imprimir mensajes como "Cargando datos...", "Guardando información..." o "Error al cargar la página".
  • Seguimiento de errores: Cuando ocurren errores en tu código, console.log() te permite mostrar mensajes de error o información específica para entender lo que está sucediendo. Esto puede ser útil para identificar y resolver problemas en tu aplicación.
  • Verificar el flujo de ejecución: Con console.log(), puedes observar el flujo de ejecución de tu código y asegurarte de que se esté comportando como esperas. Puedes colocar console.log() en diferentes partes de tu código para ver qué sucede en cada etapa.
  • Inspección de objetos y arreglos: Al imprimir objetos o arreglos con console.log(), puedes ver su contenido y estructura en la consola, lo que te facilita la inspección y comprensión de la información almacenada en ellos.
  • Medición de tiempos: console.log() también se puede utilizar para medir el tiempo de ejecución de una función o un bloque de código. Puedes utilizar console.time() y console.timeEnd() para calcular el tiempo transcurrido entre ambos puntos y así optimizar el rendimiento de tu aplicación.

Para utilizar console.log() en JavaScript, sigue estos pasos:

  • Abre tu editor de texto o entorno de desarrollo (como Visual Studio Code) y crea un nuevo archivo con extensión ".js". Por ejemplo, "mi_script.js".
  • Dentro del archivo, puedes escribir código JavaScript y usar console.log() para imprimir mensajes en la consola del navegador o en la terminal, dependiendo del entorno donde estés ejecutando el código.
  • Escribe el siguiente código de ejemplo para usar console.log(): 
                        
                            
                        // Esto es un comentario de una línea en JavaScript
                        let nombre = "Jose";
                        let edad = 30;

                        console.log("Hola, mi nombre es " + nombre + " y tengo " + edad + " años.");

¿Qué es una Variable?

En programación, una variable es un espacio de memoria reservado para almacenar datos. Estos datos pueden ser valores numéricos, cadenas de texto, objetos, arreglos o cualquier otro tipo de información que el lenguaje de programación pueda manipular.

Las variables son esenciales en la programación, ya que permiten almacenar y manipular datos de manera dinámica durante la ejecución de un programa. En lugar de tener valores fijos y constantes, las variables pueden cambiar su contenido durante la ejecución del programa, lo que le da flexibilidad y poder a la aplicación.

Para utilizar una variable, primero debes declararla, lo que significa reservar un espacio en la memoria para almacenar su valor. La sintaxis para declarar una variable varía según el lenguaje de programación que estés utilizando. Aquí hay un ejemplo en varios lenguajes: por ejemplo 

                        
        
                            let edad; // Declaración de una variable llamada "edad"

También puedes declarar varias variables en una sola línea utilizando una coma para separarlas. 

                            
                                let edad, peso; // Declaración de dos variables usando "," para separarlas.

Una vez que has declarado una variable, puedes asignarle un valor utilizando el operador de asignación "=": 

                            
                                let edad = 30; // Asignación del valor 30 a la variable "edad" en JavaScript

También puedes declarar una variable e inicializarla con una valor; 

                            
                                let edad = 35; // Creación variable y asignacion en una misma linea

A lo largo del programa, puedes modificar el valor de una variable asignándole un nuevo valor: 

                            
                                edad = 35; // Modificación del valor de la variable "edad" en JavaScript

Las variables son fundamentales para realizar cálculos, almacenar información y tomar decisiones basadas en el contenido que contienen. Es uno de los conceptos más básicos pero cruciales en la programación, y comprender cómo utilizar y manejar variables es esencial para escribir código efectivo y funcional.

Operadores

En JavaScript, existen varios tipos de operadores que te permiten realizar diferentes tipos de operaciones en tus programas. A continuación, se presentan los tipos de operadores más comunes en JavaScript:

Operadores Aritméticos

Operadores aritméticos: Se utilizan para realizar operaciones matemáticas en números.

Operadores Aritméticos
Operador Descripción Ejemplo
+ Suma 2 + 3 = 5
- Resta 5 - 2 = 3
* Multiplicación 2 * 3 = 6
/ División 6 / 2 = 3
% Módulo 7 % 3 = 1
** Potencia 2 ** 3 = 8

Ejemplos 

                            
                                //suma
                                console.log(2+3); //5
                                //resta
                                console.log(5-2); //3
                                //Multiplicación
                                console.log(2*3); //6
                                //División
                                console.log(6/2); //3
                                //Módulo
                                /*Modulo: Muestra el residuo de una división
                                En este caso 7 dividido entre 3 es igual a 2 y el
                                residuo es 1*/
                                console.log(7/3); //1
                                //Potencia
                                console.log(2 ** 3); //8

Operadores de Asignación

Un operador de asignación es un componente fundamental en la mayoría de los lenguajes de programación. Su función principal es asignar un valor a una variable. En términos simples, permite guardar un valor en una variable para que pueda ser utilizado y manipulado posteriormente en el programa.

Operadores de Asignación
Operador Descripción Ejemplo
= Asignación simple let x = 5;
+= Asignación con suma x += 3; // x = x + 3;
-= Asignación con resta x -= 2; // x = x - 2;
*= Asignación con multiplicación x *= 4; // x = x * 4;
/= Asignación con división x /= 2; // x = x / 2;
%= Asignación con módulo x %= 3; // x = x % 3;

Ejemplos 

                            
                                //Asignación simple
                                let x = 5;
                                console.log(x) //5

                                //Asignación con suma
                                let y = 5;
                                y += 3;
                                //x=x+3
                                console.log(y) //8

                                //Asignación con Multiplicación
                                let m = 5;
                                m *= 3;
                                //x=x*3
                                console.log(m) //15

                                //Asignación con División
                                let n = 5;
                                n /= 3;
                                //x=x/3
                                console.log(n) //1.666


                                //Asignación con módulo
                                let w = 5;
                                w %=3
                                console.log(w) //2

Operadores de Comparación

Operadores de comparación: Se utilizan para comparar valores y devuelven un valor booleano (true o false).

Operadores de Comparación
Operador Descripción Ejemplo
== Igual a (valor) 3 == 3 // true
!= Diferente de (valor) 3 != 4 // true
=== Igual a (valor y tipo) 3 === '3' // false
!== Diferente de (valor y tipo) 3 !== '3' // true
< Menor que 4 < 5 // true
> Mayor que 5 > 3 // true
<= Menor o igual que 3 <= 3 // true
>= Mayor o igual que 5 >= 4 // true

Ejemplos 

                            
                                //Igual a (valor)
                                console.log(3==3) //true

                                //Diferente de (valor)
                                console.log(3 != 4) //true

                                //Igual a (valor y tipo)
                                console.log(3 === '3') //false

                                //Diferente de (valor y tipo)
                                console.log(3!=='3') //true

                                //Menor que
                                console.log(4<5) //true

                                //Mayor que
                                console.log(5>3) //true

                                // 	Menor o igual que
                                console.log(3<=3) //true

                                // 	Menor o igual que
                                console.log(5>=3) //true

Operadores Lógicos

Operadores lógicos: Se utilizan para combinar o invertir valores booleanos.

Operadores Lógicos
Operador Descripción Ejemplo
&& AND lógico true && false // false
|| OR lógico true || false // true
! NOT lógico !true // false

Ejemplos 

                            
                                //&& 	AND lógico
                                console.log(true && false) //false

                                // ||  	OR lógico 
                                console.log(true || false) //true

                                // !   	NOT lógico
                                console.log(!true) //false

Operadores de Incremento y Decremento

Operadores de incremento y decremento: Se utilizan para aumentar o disminuir el valor de una variable en 1.

Operadores de Incremento y Decremento
Operador Descripción Ejemplo
++ Incremento x++; // Incrementa el valor de x en 1
-- Decremento x--; // Decrementa el valor de x en 1

Ejemplo 

                            
                                // ++ Incremento 
                                let x = 3;
                                x++;
                                console.log(x); // 4 Incrementa el valor de x en 1

                                // -- Decremento
                                let y = 10;
                                y--;
                                console.log(y); // 9 Decrementa el valor de x en 1

Operadores de Concatenación

Operadores de concatenación: Se utilizan para unir cadenas de texto (strings). +: Concatenación de cadenas

Operadores de Concatenación
Operador Descripción Ejemplo
+ Concatenación de cadenas 'Hola ' + 'Mundo' // 'Hola Mundo'

Ejemplos 

                            
                           
                               
                                console.log("Hola " + "Mundo"); //Hola Mundo
