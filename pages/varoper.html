Variables y Tipos de Operadores

¿Qué es una Variable?

En programación, una variable es un espacio de memoria reservado para almacenar datos. Estos datos pueden ser valores numéricos, cadenas de texto, objetos, arreglos o cualquier otro tipo de información que el lenguaje de programación pueda manipular.

Las variables son esenciales en la programación, ya que permiten almacenar y manipular datos de manera dinámica durante la ejecución de un programa. En lugar de tener valores fijos y constantes, las variables pueden cambiar su contenido durante la ejecución del programa, lo que le da flexibilidad y poder a la aplicación.

Para utilizar una variable, primero debes declararla, lo que significa reservar un espacio en la memoria para almacenar su valor. La sintaxis para declarar una variable varía según el lenguaje de programación que estés utilizando. Aquí hay un ejemplo en varios lenguajes: por ejemplo 

                  
  
                    let edad; // Declaración de una variable llamada "edad"

También puedes declarar varias variables en una sola línea utilizando una coma para separarlas. 

                    
                        
                        let edad, peso; // Declaración de dos variables usando "," para separarlas.

Una vez que has declarado una variable, puedes asignarle un valor utilizando el operador de asignación "=": 

                      
                          
                          let edad = 30; // Asignación del valor 30 a la variable "edad" en JavaScript

También puedes declarar una variable e inicializarla con una valor; 

                        
                            let edad = 35; // Creación variable y asignacion en una misma linea

A lo largo del programa, puedes modificar el valor de una variable asignándole un nuevo valor: 

                        
                            edad = 35; // Modificación del valor de la variable "edad" en JavaScript

Las variables son fundamentales para realizar cálculos, almacenar información y tomar decisiones basadas en el contenido que contienen. Es uno de los conceptos más básicos pero cruciales en la programación, y comprender cómo utilizar y manejar variables es esencial para escribir código efectivo y funcional.

Operadores

En JavaScript, existen varios tipos de operadores que te permiten realizar diferentes tipos de operaciones en tus programas. A continuación, se presentan los tipos de operadores más comunes en JavaScript:

Operadores Aritméticos

Operadores aritméticos: Se utilizan para realizar operaciones matemáticas en números.

Operadores Aritméticos
Operador Descripción Ejemplo
+ Suma 2 + 3 = 5
- Resta 5 - 2 = 3
* Multiplicación 2 * 3 = 6
/ División 6 / 2 = 3
% Módulo 7 % 3 = 1
** Potencia 2 ** 3 = 8

Ejemplos 

                    
                        //suma
                        console.log(2+3) //5
                        //resta
                        console.log(5-2) //3
                        //Multiplicación
                        console.log(2*3) //6
                        //División
                        console.log(6/2) //3
                        //Módulo
                        /*Modulo: Muestra el residuo de una división
                        En este caso 7 dividido entre 3 es igual a 2 y el
                        residuo es 1*/
                        console.log(7/3) //1
                        //Potencia
                        console.log(2 ** 3) //8

Operadores de Asignación

Operadores de Asignación
Operador Descripción Ejemplo
= Asignación simple var x = 5;
+= Asignación con suma x += 3; // x = x + 3;
-= Asignación con resta x -= 2; // x = x - 2;
* Asignación con multiplicación x *= 4; // x = x * 4;
/= Asignación con división x /= 2; // x = x / 2;
%= Asignación con módulo x %= 3; // x = x % 3;

Operadores de Comparación

Operadores de comparación: Se utilizan para comparar valores y devuelven un valor booleano (true o false).

Operadores de Comparación
Operador Descripción Ejemplo
== Igual a (valor) 3 == 3 // true
!= Diferente de (valor) 3 != 4 // true
=== Igual a (valor y tipo) 3 === '3' // false
!== Diferente de (valor y tipo) 3 !== '3' // true
< Menor que 4 < 5 // true
> Mayor que 5 > 3 // true
<= Menor o igual que 3 <= 3 // true
>= Mayor o igual que 5 >= 4 // true

Operadores Lógicos

Operadores lógicos: Se utilizan para combinar o invertir valores booleanos.

Operadores Lógicos
Operador Descripción Ejemplo
&& AND lógico true && false // false
|| OR lógico true || false // true
! NOT lógico !true // false

Operadores de Incremento y Decremento

Operadores de incremento y decremento: Se utilizan para aumentar o disminuir el valor de una variable en 1.

Operadores de Incremento y Decremento
Operador Descripción Ejemplo
++ Incremento x++; // Incrementa el valor de x en 1
-- Decremento x--; // Decrementa el valor de x en 1

Operadores de Concatenación

Operadores de concatenación: Se utilizan para unir cadenas de texto (strings). +: Concatenación de cadenas

Operadores de Concatenación
Operador Descripción Ejemplo
+ Concatenación de cadenas 'Hola ' + 'Mundo' // 'Hola Mundo'
